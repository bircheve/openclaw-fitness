# Barbell Row

Primary back thickness builder and foundational horizontal pull.

## Why This Exercise

- Builds overall back thickness (lats, rhomboids, traps)
- Heavy compound movement for progressive overload
- Strengthens erectors and core isometrically
- Carryover to deadlifts and other pulls

## Row Variations by Torso Angle

| Angle | Emphasis |
|-------|----------|
| 45 degrees | Standard, balanced |
| Near horizontal | More lat focus, harder |
| More upright (60-70) | More traps/rhomboids, easier |

## Grip Options

| Grip | Effect |
|------|--------|
| Overhand (pronated) | Standard, balanced |
| Underhand (supinated) | More lat emphasis, more bicep |
| Wide grip | More rear delt and upper back |
| Close grip | More lat thickness |

## Form Priority

This exercise is easy to cheat on. Don't.

- If you're standing up to complete the row, it's too heavy
- Control the eccentric - don't drop the weight
- Squeeze at the top for 1 second
- Keep your ego in check and use weight you can control

## Common Programming

- 3-4 sets of 8-10 reps
- Can go heavier than pulldowns (fewer reps)
- Often programmed after vertical pulls on pull day

## Variations

- Pendlay Row (from floor each rep, more explosive)
- T-Bar Row (different grip, machine supported)
- Seated Cable Row (constant tension)
- Dumbbell Row (unilateral)
