# Reference Programs

Historical workout templates. Draw from these for creative exercise selection, set/rep schemes, and intensity techniques when generating fresh workouts during program rotation.

**How to use this file**: When generating a new 2-week program block, pull exercises, techniques, rep schemes, and set structures from both programs below. Mix and match — don't copy a full day verbatim. The goal is variety and progression, not repetition.

## Training Definitions

These techniques appear throughout the reference programs. Use these techniques when building workouts.

**Drop Set**
On the last set, complete the prescribed reps, then immediately drop to a lighter weight and complete more reps with no rest between.
Example notation: `12, 10, 8 drop 8` — pyramid up in weight for 3 sets, then drop set 8 reps on the final set.

**Double Drop Set**
Same as a drop set but drop the weight twice.
Example notation: `10, 8, 6 drop 8 drop 10` — after the last set, drop weight for 8 reps, drop again for 10 reps.

**Superset / Compound Set**
Complete reps of the first exercise, then immediately do reps of a second exercise within the same set — no rest between.
Example notation: `4 sets of 10/15` — 10 reps exercise A, 15 reps exercise B, that's one set.

**Pyramid Sets**
Reps decrease each set (e.g., 12, 10, 8, 8). Increase weight each set as reps go down.

**Straight Sets**
Same reps every set (e.g., 8, 8, 8, 8). Tempo-based — moving up in weight isn't necessary. Focus on control and contraction.

**Failure**
No prescribed rep count — perform reps until muscular failure.
Example notation: `4 sets to failure`

**Pause Reps**
Hold a specific position (stretch or contraction) for a prescribed duration during each rep.
Example notation: `*2 second pause at the bottom of each rep`

**Slow Negatives**
Extended eccentric (lowering) phase for time under tension.
Example notation: `*10 second negatives per rep`
