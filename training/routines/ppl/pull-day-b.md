# Pull Day B - Back Thickness

Row-focused pull day for building back thickness and mid-back detail.

## Session Flow

1. **Warmup** (10-15 min)
2. **Lifting** (~75 min): Execute exercises in order
3. **Cardio Finisher** (35 min): Stairmaster at moderate intensity

## Exercise Order

| # | Exercise | Sets | Reps | Rest | Notes |
|---|----------|------|------|------|-------|
| 1 | Barbell Row | 4 | 8-10 | 60s | Primary row |
| 2 | Seated Cable Row | 4 | 10-12 | 60s | Constant tension |
| 3 | Lat Pulldown | 3 | 10-12 | 60s | Width maintenance |
| 4 | Dumbbell Shrug | 3 | 12-15 | 45s | Trap development |
| 5 | Face Pull | 3 | 15-20 | 45s | Rear delt/upper back |
| 6 | Incline Dumbbell Curl | 3 | 10-12 | 45s | Long head stretch |
| 7 | Barbell Curl | 3 | 10-12 | 45s | Drop set on last |

## Execution Notes

- Rows are the priority - squeeze hard at contraction
- All movements use 3-5 second eccentrics
- Last set of each exercise to failure
- Use straps on rows if grip limits back work

## Total Volume

- Back: 14 sets
- Traps: 3 sets
- Biceps: 6 sets

## Thickness vs. Width

Pull Day A focuses on width (vertical pulls, lats).
Pull Day B focuses on thickness (horizontal pulls, mid-back).

Both are needed for complete back development.

## Substitutions

If equipment unavailable:
- Barbell Row → T-Bar Row
- Seated Cable Row → Machine Row
- Dumbbell Shrug → Barbell Shrug

## Row Technique

For thickness:
- Pull to lower chest/upper abdomen
- Squeeze shoulder blades together hard
- Hold the contraction for 1 second
- Control the eccentric - no bouncing

## Intra-Workout

Sip throughout lifting:
- 25g whey protein
- 10g BCAAs
- 5g glutamine

## Post-Workout

Within 10 minutes of cardio completion:
- 40g whey protein
- 32oz electrolyte drink
