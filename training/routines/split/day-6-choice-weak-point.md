# Day 6 — Choice / Weak Point

## Target Muscles
- **Varies** — whatever feels undertrained this week

## Common Choices
- **Arms day**: Biceps + Triceps superset session
- **Rear delts + calves + abs**: Accessory catch-up
- **Full upper pump**: Light, high-rep chest/back/shoulders
- **Extra legs**: If Day 2 felt insufficient

## Block Structure

### Phase 2 — Primary Compound
- 1-2 compound movements for the chosen focus
- Sets: 3 each, reps: 8-12, rest: 90-120 sec
- Can be skipped if the day is pure isolation work

### Phase 3 — Secondary Hypertrophy
- 3-4 isolation movements for the chosen focus
- Sets: 3 each, reps: 12-15, rest: 45-75 sec
- Supersets encouraged to keep session short

### Phase 4 — Finisher
- Optional — depends on energy and time

### Phase 5 — Cardio
- **Optional** — do 20-30 min if energy allows, skip if fatigued
