# Training Protocol

The authoritative reference for the user's current training program. The coach reads this every time a workout is requested.

## High-Level Constants

| Parameter | Value |
|-----------|-------|
| Days/Week | 7 (Day 7 is active recovery cardio) |
| Session Length | 60-90 min total (lifting + cardio) |
| Priority | Tempo and control > load |
| Volume | High — 10-16 working sets per muscle group per session |
| Rest Periods | 45-75 sec (isolation), 90-120 sec (heavy compounds) |
| Rep Ranges | 8-12 compounds, 12-15 isolation |
| Eccentrics | 3-5 seconds on every rep |
| Failure | Last 1-2 sets of each exercise to failure or near-failure |
| Intensity | Push hard — every session should feel earned |

## Equipment

This program assumes access to a full commercial gym with free weights, machines, cables, and cardio equipment. Adjust exercise selections if equipment is limited.

## Mindset

Push hard. Every set should have purpose. Don't coast — if the weight feels easy, increase it or slow down the tempo. The goal is to leave the gym knowing you gave it everything. Comfortable workouts don't build muscle.

## Session Template

Every workout follows 5 phases:

### Phase 1 — Warm-up (10-15 min)
- 5 min low-intensity cardio (bike or incline walk)
- Dynamic stretching for target muscle groups
- 1-2 feeder sets on the first compound movement (50-60% working weight)

### Phase 2 — Primary Compound Block (20-30 min)
- 2-3 compound movements for the day's primary muscle groups
- 3-4 working sets each
- Rep range: 8-12
- Rest: 90-120 sec
- Focus: progressive overload, controlled eccentrics
- Last set to failure

### Phase 3 — Secondary Hypertrophy Block (20-30 min)
- 3-4 isolation or machine movements
- 3 working sets each
- Rep range: 12-15
- Rest: 45-75 sec
- Techniques: drop sets on final set, supersets where noted
- Mind-muscle connection is the priority

### Phase 4 — Finisher (5-10 min)
- 1-2 burnout movements targeting the day's primary muscles
- High reps (15-20+), light weight, maximum pump
- Options: cable work, bands, bodyweight, giant sets
- Don't skip this — it's where the pump happens and the session gets pushed from good to great

### Phase 5 — Cardio (20-40 min)
- Default: 30-40 min stairmaster at moderate intensity (130-150 BPM)
- Day-specific overrides:
  - After leg day: **no cardio** (legs are fried)
  - Choice/weak-point day: cardio is optional
  - Day 7 (active recovery): **75 min moderate cardio, no lifting** — Zone 2, 120-140 BPM
  - All other days: cardio required
- Can substitute: incline treadmill walk, rowing machine

| Day 6 | Choice / Weak Point | Optional |
| Day 7 | Active Recovery Cardio | Yes — 75 min stairmaster |

Notes:
- The split is a **rolling 7-day cycle**, not calendar-fixed. Day 1 starts whenever the last cycle ended.
- If a day is missed, pick up where you left off — don't skip ahead.
- Day 6 targets whatever muscle group feels undertrained that week. Common choices: arms, rear delts, calves, abs.
- There are no full rest days. Day 7 is active recovery (cardio only, no lifting).

## Exercise Selection Rules

Exercises are chosen by **category**, not fixed. The coach selects from the exercise library or general knowledge based on these rules:

### Per-session categories

| Day | Compound categories | Isolation categories |
|-----|--------------------|--------------------|
| Day 1 (Chest + Biceps) | Flat press, incline press | Chest fly, cable crossover, bicep curl (barbell or dumbbell), hammer curl |
| Day 2 (Legs + Side Delts) | Squat variation, leg press, Romanian deadlift | Leg extension, leg curl, lateral raise, calf raise |
| Day 3 (Back + Triceps) | Row variation, pull-up/pulldown | Cable row, rear delt fly, tricep pushdown, overhead tricep extension |
| Day 4 (Shoulders + Legs light) | Overhead press, front squat or leg press (moderate) | Lateral raise, face pull, rear delt fly, leg extension (light) |
| Day 5 (Chest + Upper Push) | Incline press, dip variation | Cable fly, tricep isolation, front raise |
| Day 6 (Choice) | Varies | Varies |

### Rotation rules
- **Rotate specific exercises every 2-3 weeks** within each category. For example, swap barbell incline press for dumbbell incline press, or cable fly for pec deck.
- **Vary angles and grips**: if last session used flat barbell bench, this session might use slight decline dumbbell press.
- **Equipment variety**: alternate barbell ↔ dumbbell ↔ machine ↔ cable for the same movement pattern.
- **Never repeat the exact same exercise lineup** two sessions in a row for the same muscle group.

## Progression Model — Density-Based

The user does **not** use traditional linear load progression (add 5 lbs every week). Instead, progression is density-based:

### Progression levers (in order of priority)
1. **More reps at the same weight** — if you got 10 last time, aim for 11-12
2. **Slower eccentrics** — extend the lowering phase from 3 sec to 4-5 sec
3. **Pauses** — add a 1-2 sec pause at the stretch position
4. **Reduced rest** — cut rest periods by 10-15 sec
5. **Cleaner failure** — reach failure with better form than last time
6. **Load increase** — only after exhausting the above for 2-3 weeks on a given exercise

### When to increase weight
- All working sets hit the **top** of the rep range (e.g., 12 on a 8-12 exercise) for **2 consecutive sessions**
- Increase by the smallest available increment (5 lbs barbell, 2.5 lbs dumbbell)
- After increasing, expect reps to drop to the bottom of the range — that's normal

### Deload
- Every 5-6 weeks, or when multiple exercises stall simultaneously
- Reduce volume by 40% (fewer sets, not fewer exercises)
- Keep intensity moderate — no failure training during deload week
- Duration: 1 week
